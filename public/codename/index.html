Install Codename on MS Windows

  1. Download our windows installer from here
  2. Run the installer to whitelist the extension in chrome and .
  3. Download the extension from here.
  4. Unzip the extension.zip, and drag and drop the .crx file to chrome://extensions page in Google Chrome and/or Brave. NOTE: Turn on Developer Mode.

Usage

Follow the installation guide and fill in your profile.

After filling in your profile, the extension will start to fetch jobs in the background. The initial fetch may take a few seconds, but subsequent fetches will be faster. Dont forget to pin the extension for quick access.

After filling in your profile, you should be able to use the following bindings on whatever website. The extension will fill in the corresponding profile attributes for you.

Binding Filled attribute
/f First name
/l Last name
/e Email
/p Phone
/in LinkedIn
/git GitHub
/w Website
/a0 Address line 1
/a1 Address line 2
/at Address city
/ap Address province
/ac Address country
/ax Address postal code
